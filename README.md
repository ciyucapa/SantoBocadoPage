# 🥐 Santo Bocado

Landing page moderna para **Santo Bocado**, una panadería artesanal especializada en croissants, masa madre, babkas y productos horneados frescos diariamente ✨

## 🌟 Vista General

Santo Bocado nace del amor por la panadería artesanal y los productos hechos con dedicación.  
Esta página fue desarrollada para presentar la marca de manera elegante, cálida y profesional, permitiendo a los clientes conocer los productos y realizar pedidos fácilmente vía WhatsApp.

---

## ✨ Características

- 🥐 Diseño moderno y artesanal
- 📱 Responsive para móvil, tablet y desktop
- ⚡ Desarrollado con Next.js
- 🎨 Estilizado con Tailwind CSS
- 📸 Galería y productos visuales
- 💬 Integración directa con WhatsApp
- 📄 Descarga de catálogo PDF
- 🌙 Secciones premium:
  - Hero
  - Quiénes Somos
  - Productos
  - Testimonios
  - FAQ
  - Footer personalizado

## 🚀 Instalación

Clona el proyecto:

```bash
git clone https://github.com/TU-USUARIO/santo-bocado.git
```

Entra a la carpeta:

```bash
cd santo-bocado
```

Instala dependencias:

```bash
npm install
```

Ejecuta el proyecto:

```bash
npm run dev
```
